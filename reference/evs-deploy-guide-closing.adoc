= 结束语
:allow-uri-read: 


[role="lead"]
当配置为使用 Amazon EVS 和Amazon FSx for NetApp ONTAP作为 DR 目标时， NetApp Disaster Recovery可为您的内部部署 VMware 基础架构提供经济高效的混合云 DR 解决方案。这为您提供了针对关键任务基础设施的强大 DR 目标。  NetApp Disaster Recovery利用原生ONTAP数据保护功能，这些功能以数十年的数据保护经验为后盾，经过深入测试，可确保您的数据得到全面保护。
