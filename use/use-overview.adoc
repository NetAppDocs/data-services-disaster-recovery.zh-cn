= 使用NetApp Disaster Recovery概述
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
使用NetApp Disaster Recovery，您可以实现以下目标：

* link:../use/dashboard-view.html["查看灾难恢复计划的健康状况"]
* link:../use/sites-add.html["向站点添加资源"]
* link:../use/resource-group.html["创建资源组以组织数据源"]
* link:../use/drplan-create.html["制定灾难恢复计划"]
* link:../use/replicate.html["复制 VMware 应用程序"] 从主站点到云中的灾难恢复远程站点，使用 SnapMirror 复制
* link:../use/migrate.html["迁移应用程序"] 从您的主站点到另一个站点
* link:../use/failover.html["测试故障转移"] 不中断资源
* 在发生灾难时，link:../use/failover.html["故障转移您的主站点"]
* 灾难解决后，link:../use/failback.html["故障恢复"]从灾难恢复站点到主站点
* link:../use/monitor-jobs.html["监控灾难恢复操作"] 在作业监控页面上
